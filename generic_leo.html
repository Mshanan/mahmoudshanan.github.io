Leo Rover

The aim of this project is to develop a mobile robot (Leo Rover), which can autonomously retrieve an object from an environment.
This is an ongoing project, which is part of my MSc Robotic System Design unit.

• The Leo rover is a stable and customisable mobile robot, that can be deployed indoors as well as outdoors, and withstand any environment.
• The original Leo Rover will be upgraded; a loading cell will be designed, 3-D printed, and placed on top of the rover to fit a robotic manipulator, stereo and thermal cameras, a Lidar, and IMUs.
• The Leo Rover will be initially programmed such that it is steered through a joystick or a keyboard.
• The Leo Rover software will then include PID controllers, and machine learning and computer vision algorithms such that it is completely autonomous. It will be designed and implemented using ROS 2, Gazebo simulation, python, and C++.
