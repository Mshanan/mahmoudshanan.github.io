Robot Vaccum Cleaner Simulation (ROS 2, Gazebo)

  • Developed the URDF model of a robot vaccum cleaner, including a lidar, depth camera, and IMU (using the xacro package), and visualised the model in Rviz.
  • Utilised the ROS 2 "Nav2" and "behaviortree" packages such that the robot navigates autonomously using SLAM, A* path planning algorithm, and behaviour trees.
  • Implemented the "twist_mux" package such that the robot could be also controlled manually using the keyboad, and the system was tested on Gazebo.
